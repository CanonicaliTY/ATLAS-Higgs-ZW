# ATLAS Open Data Analysis Workspace

The actively maintained Manchester 3rd-year `Z -> ll` lab analysis lives in:

`Analysis code/my-ATLAS-code/ATLAS-parquet/`

Start there:

- project README: `Analysis code/my-ATLAS-code/ATLAS-parquet/README.md`
- main entry point: `Analysis code/my-ATLAS-code/ATLAS-parquet/main.py`
- legacy wrapper: `Analysis code/my-ATLAS-code/ATLAS-parquet/RearrangedCode_v2.0.py`

That analysis keeps the modular refactor, the tight-parquet workflow, the order-invariant control-region masks, the significance-based isolation optimisation scan, and the new cut-consistent DD-background evaluation pipeline.

The current outputs are written under:

`Analysis code/my-ATLAS-code/ATLAS-parquet/output_py/`

If you are looking for the current physics logic or run instructions, use the analysis README rather than older generated outputs in `output_py/`.
